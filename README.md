# slider-for-fsd

Empty project.

## Building and running on localhost

First install dependencies:

```sh
npm install
```

Run webpack dev server:

```sh
npm run dev
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build-dev
```

## Running

```sh
node dist/bundle.js
```

## Testing

To run unit tests:

```sh
npm test
```
